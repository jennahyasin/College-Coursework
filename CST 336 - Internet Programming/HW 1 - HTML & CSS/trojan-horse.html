TROJAN HORSE VIRUS:

What are Trojan Horse Viruses?
  • A malware that intentionally disguises itself as an actual program through a user's software.
  • The name Trojan Horse comes from a peom called Odyssey, by Homer. The horse disguises himself as a gift in order to be let through the gates of the city in this poem.
  • In terms of the virus, it can be thought of as how the virus disguises itself in either a link or in a file and once it is clicked by a user, unknowingly of what is about to happen, the attacker's invasive code is downloaded and it runs through that code on the user's device.
